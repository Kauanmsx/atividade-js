README - Atividade de Introdução ao JavaScript
Objetivo

Esta atividade tem como objetivo fornecer uma introdução ao JavaScript, ensinando conceitos básicos como variáveis, operações matemáticas, estruturas condicionais e outras funcionalidades fundamentais da linguagem. Cada exercício aborda uma funcionalidade específica para consolidar o aprendizado.

Descrição dos Exercícios

Exibição de uma mensagem na tela

O código imprime no console a mensagem "Olá, mundo! Bem-vindo ao JavaScript."

Exibição de variáveis usando template literals

Definindo variáveis de nome, idade e cidade, o programa exibe uma mensagem personalizada utilizando template literals para concatenar as variáveis com a string.

Operação de soma entre dois números

O código realiza a soma de duas variáveis e imprime o resultado no console.

Operações matemáticas básicas

Realiza a soma, subtração, multiplicação e divisão de dois números e imprime os resultados no console.

Verificação de maioridade

Com base na variável idade, o código verifica se a pessoa é maior ou menor de idade e exibe a resposta no console.

Identificação de número positivo, negativo ou zero

A variável número é verificada para identificar se é positivo, negativo ou igual a zero, exibindo o resultado correspondente.

Cálculo do IMC (Índice de Massa Corporal)

Calcula o IMC com base no peso e altura fornecidos e exibe o resultado com a categoria de peso (abaixo do peso, peso normal, sobrepeso ou obesidade).

Encontrar o maior número entre três números

Compara três números e exibe o maior deles.

Conversão de temperatura de Celsius para Fahrenheit

Converte uma temperatura de Celsius para Fahrenheit e exibe o resultado no console.

Verificação de número par ou ímpar

Verifica se o número fornecido é par ou ímpar e exibe o resultado no console.

Cálculo da média de notas

Calcula a média de três notas e exibe se o aluno foi aprovado, está em recuperação ou reprovado, com base no valor da média.

Cálculo do antecessor, número e sucessor

Exibe o número fornecido, seu antecessor e seu sucessor no console.

Conclusão

Esta atividade tem como objetivo apresentar conceitos fundamentais do JavaScript de forma prática. Através desses exercícios, você pode aprender como trabalhar com variáveis, operações matemáticas, condicionais e cálculos simples, que são a base para programar aplicações mais complexas.
